## Naming convention

-   Filename:
    -   Component: `/realtime-overview/ListPanel.tsx`
    -   Other: `/mock-api/realtime-overview.ts`
-   URL:
    -   Path: `/realtime-overview/user-acquisition/top`
    -   Query: `dimensionName=value&page=1`
-   JSON Key: `reportHour`, `userCount`
